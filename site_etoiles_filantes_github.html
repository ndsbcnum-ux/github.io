Ce que tu peux en faire

Ce modèle peut servir de site vitrine, portfolio, page de présentation de service, page personnelle ou mini site de projet. Il fonctionne très bien sur ordinateur comme sur téléphone.

Présentation

Ajoute ton nom, ton activité, un texte de présentation et un bouton vers tes réseaux ou ton mail.

Projet

Explique une idée, montre tes services ou décris un projet personnel avec un visuel propre.

Contact

Tu peux intégrer un bouton vers ton e-mail, Telegram, Instagram, TikTok ou tout autre lien.

Un design simple à modifier

Tous les textes ici sont des exemples. Tu peux tout remplacer directement dans le fichier HTML : le titre, les couleurs, les boutons, les sections et même la vitesse des étoiles filantes.

Astuce : pour publier ce site sur GitHub Pages, crée un dépôt GitHub, ajoute ce fichier sous le nom index.html, puis active GitHub Pages dans les paramètres du dépôt. Le site sera ensuite accessible en ligne avec une adresse du type tonpseudo.github.io.
